# Social Media Feed (Dark Mode Project)

A responsive and dynamic social media feed web app inspired by Instagram and Twitter — built using **HTML**, **CSS**, and **JavaScript**.  
It features dark/light theme toggling, dynamic posts, image previews, infinite scrolling, and interactive post actions.

---

## Features

- **Dark/Light Theme Toggle** — Switch themes instantly, auto-applies across all sections.  
- **Post Creation** — Create text or image posts dynamically.  
- **Live Image Preview** — See your post image before uploading.  
- **Infinite Scroll** — New posts load automatically as you scroll down.  
- **Like, Comment & Share** — Interact with posts in real time.  
- **Inline Comments** — Add and view comments under posts dynamically.  
- **Clean, Modern UI** — Styled to match the website’s dark theme design.
